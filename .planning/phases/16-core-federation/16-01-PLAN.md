  <action>
Extend PeerCache in src/federation/cache.rs with networking capabilities. Per decisions D-01 through D-07:

1. Add `client: reqwest::Client` field to PeerCache struct. Update `PeerCache::new()` to create the client:
   ```rust
   client: reqwest::Client::builder()
       .timeout(std::time::Duration::from_secs(10))
       .build()
       .expect("Failed to build HTTP client"),
   ```

2. Add `fetch_peer(&self, pubkey: &str)` async method:
   - Acquire write lock on peers
   - Get the peer's URL from the HashMap
   - Release write lock before HTTP call
   - GET `{url}/registry` using self.client
   - On success: parse response body as PeerRegistry, acquire write lock, update peer's registry to Some(parsed), set last_success to Some(Instant::now()), set last_attempt to Some(Instant::now()), set status to PeerStatus::Fresh
   - On failure (reqwest error or parse error): log WARN with peer pubkey and error, acquire write lock, set last_attempt to Some(Instant::now()), keep existing registry. If last_success is None or last_success is Some(t) where t.elapsed() > Duration::from_secs(3600), set status to PeerStatus::Stale (if registry exists) or PeerStatus::Unreachable (if no registry)

3. Add `refresh_all(&self)` async method:
   - Acquire read lock, collect Vec of pubkey strings, release lock
   - Iterate sequentially (per D-06): for each pubkey, call self.fetch_peer(&pubkey).await
   - Log info at start: "Refreshing peer cache ({N} peers)"

4. Add `get_all_cached(&self) -> Vec<CachedPeerSnapshot>` async method:
   - Define a new struct at module level:
     ```rust
     #[derive(Debug, Clone)]
     pub struct CachedPeerSnapshot {
         pub pubkey: String,
         pub url: String,
         pub name: Option<String>,
         pub registry: Option<PeerRegistry>,
         pub stale: bool,
         pub status: PeerStatus,
     }
     ```
   - Acquire read lock, iterate peers, create snapshots where stale = (status == PeerStatus::Stale)
   - Return Vec of snapshots

5. Update the `pub use` in mod.rs to export CachedPeerSnapshot if needed.

6. Add `use std::time::{Duration, Instant};` and `use super::types::PeerRegistry;` to imports. Add `use super::error::FederationError;` (may be used for structured logging).
