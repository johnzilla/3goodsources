Add this block at the very top of `async fn main()`, BEFORE `let config = Config::load()?;`:

```rust
// Check for fork subcommand before loading config (no env vars needed)
{
    let args: Vec<String> = std::env::args().collect();
    if args.len() > 1 && args[1] == "fork" {
        match crate::fork::run(args) {
            Ok(()) => std::process::exit(0),
            Err(e) => {
                eprintln!("Error: {}", e);
                std::process::exit(1);
            }
        }
    }
}
```

Note: This is inside `async fn main()` which has `#[tokio::main]`, but `fork::run()` is synchronous and `std::process::exit(0)` terminates before any async work happens. This is intentional per D-03 — the fork path never uses the tokio runtime for actual async work.
