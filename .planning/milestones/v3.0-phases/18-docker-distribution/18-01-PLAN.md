   - registry: `ghcr.io`
   - username: `${{ github.actor }}`
   - password: `${{ secrets.GITHUB_TOKEN }}`

5. `docker/metadata-action@v5` (per D-08):
   - id: `meta`
   - images: `ghcr.io/johnzilla/3goodsources` (per D-06)
   - tags (per D-07):
     - `type=raw,value=latest,enable={{is_default_branch}}`
     - `type=sha,prefix=`
     - `type=semver,pattern={{version}}`
     - `type=semver,pattern={{major}}.{{minor}}`

6. `docker/build-push-action@v6` (per D-04):
   - context: `.`
   - platforms: `linux/amd64,linux/arm64` (per D-03)
   - push: `true`
   - tags: `${{ steps.meta.outputs.tags }}`
   - labels: `${{ steps.meta.outputs.labels }}`
   - cache-from: `type=gha`
   - cache-to: `type=gha,mode=max`

Do NOT modify the Dockerfile (per D-09, D-10).
