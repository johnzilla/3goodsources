  <files>src/identity/mod.rs, src/identity/types.rs, src/identity/error.rs, src/identity/loader.rs, src/lib.rs</files>
  <behavior>
    - Identity deserializes from JSON with name, identity_type (serde renamed from "type"), platforms, operator_pubkey
    - IdentityType enum: Human, Bot (serde rename_all lowercase)
    - Platform enum: X, Nostr, Github (serde rename_all lowercase)
    - PlatformClaim has platform, handle, proof_url fields
    - Identity has #[serde(default)] for schema evolution
    - operator_pubkey is Option<String> with skip_serializing_if = "Option::is_none"
    - load() deserializes HashMap<String, Identity> from JSON file
    - load() validates bot identities: operator_pubkey must reference existing human identity
    - load() rejects bot with missing operator_pubkey (MissingOperator error)
    - load() rejects bot with operator_pubkey pointing to non-existent identity (InvalidOperator error)
    - load() rejects bot with operator_pubkey pointing to another bot (InvalidOperator error)
    - load() accepts empty JSON object ({}) as valid empty HashMap
    - load() returns FileRead error for missing file
    - load() returns JsonParse error for invalid JSON
  </behavior>
  <action>
    Create src/identity/ directory with 4 files following the exact audit module pattern:

    **error.rs:** IdentityError enum with thiserror derive. Variants: FileRead { path, error }, JsonParse { path, error }, InvalidOperator { pubkey, operator_pubkey }, MissingOperator { pubkey }.

    **types.rs:**
    - Identity struct: name (String), identity_type (IdentityType) with #[serde(rename = "type")], platforms (Vec<PlatformClaim>), operator_pubkey (Option<String>) with #[serde(skip_serializing_if = "Option::is_none")]. Apply #[serde(default)] on struct. Implement Default with identity_type: Human, empty platforms, None operator.
    - IdentityType enum: Human, Bot. Derive PartialEq. #[serde(rename_all = "lowercase")].
    - Platform enum: X, Nostr, Github. Derive PartialEq. #[serde(rename_all = "lowercase")].
    - PlatformClaim struct: platform (Platform), handle (String), proof_url (String).
    - All structs derive Debug, Clone, Serialize, Deserialize.

    **loader.rs:** Async load() function taking impl AsRef<Path>. Read file with tokio::fs::read_to_string. Deserialize to HashMap<String, Identity>. Validate: for each identity where identity_type == Bot, check operator_pubkey is Some and references an existing human identity. Return appropriate IdentityError variants on failure.

    **mod.rs:** pub mod error, loader, types. Re-export IdentityError, load, Identity, IdentityType, Platform, PlatformClaim.

    Add `pub mod identity` to src/lib.rs.

    Write unit tests in loader.rs covering: valid human identity, valid bot with human operator, bot missing operator_pubkey, bot with non-existent operator, bot with bot operator, empty file, invalid JSON, file not found.
